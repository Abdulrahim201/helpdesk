# helpdesk
A web-based helpdesk application built with Django and Django REST Framework.  The system allows users to submit support tickets, track their status, and  receive updates throughout the resolution process. Support agents can manage,  prioritise, and close incidents while maintaining a full audit trail.

## Features
- Ticket submission and management
- Priority levels (Low, Medium, High, Critical)
- User roles (Admin, Support Agent, Customer)
- Ticket history and audit logs
- Dashboard with ticket statistics
- Email notifications on ticket updates
- REST API with Django REST Framework

## Tech Stack
- Backend: Django, Django REST Framework
- Database: PostgreSQL
- Version Control: Git
